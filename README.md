# Projeto-dicionario

 **Projeto em desenvolvimento**
 
 Este será um aplicativo de dicionário desenvolvido com **React**, com o objetivo de permitir a busca de palavras em inglês e a visualização de seus significados utilizando uma API externa.

A aplicação poderá ser utilizada por alunos para:
- Consultar significados de palavras em inglês
- Ouvir a pronúncia correta
- Ampliar o vocabulário de forma autônoma
- Apoiar atividades propostas em sala de aula
  
Além disso, o projeto também tem como finalidade o aprimoramento de habilidades em **React** e consumo de **APIs**, unindo tecnologia e educação.

## Tecnologias Utilizadas

- React
- JavaScript
- HTML5
- CSS3
- Free Dictionary API
